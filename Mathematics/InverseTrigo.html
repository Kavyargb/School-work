Inverse Trigonometric Functions

Exercise 2.1

Find the principal values of the following:

1. \(\sin^{-1} \left(-\frac{1}{2}\right)\)

Let \(y = \sin^{-1} \left(-\frac{1}{2}\right)\).

Then \(\sin y = -\frac{1}{2}\).

The principal value branch of \(\sin^{-1} x\) is \([-\frac{\pi}{2}, \frac{\pi}{2}]\).

We know that \(\sin \left(\frac{\pi}{6}\right) = \frac{1}{2}\).

Therefore, \(\sin \left(-\frac{\pi}{6}\right) = -\frac{1}{2}\).

Since \(-\frac{\pi}{6} \in [-\frac{\pi}{2}, \frac{\pi}{2}]\), the principal value is \(-\frac{\pi}{6}\).

$$ \sin^{-1} \left(-\frac{1}{2}\right) = -\frac{\pi}{6} $$

2. \(\cos^{-1} \left(\frac{\sqrt{3}}{2}\right)\)

Let \(y = \cos^{-1} \left(\frac{\sqrt{3}}{2}\right)\).

Then \(\cos y = \frac{\sqrt{3}}{2}\).

The principal value branch of \(\cos^{-1} x\) is \([0, \pi]\).

We know that \(\cos \left(\frac{\pi}{6}\right) = \frac{\sqrt{3}}{2}\).

Since \(\frac{\pi}{6} \in [0, \pi]\), the principal value is \(\frac{\pi}{6}\).

$$ \cos^{-1} \left(\frac{\sqrt{3}}{2}\right) = \frac{\pi}{6} $$

3. \(\csc^{-1} (2)\)

Let \(y = \csc^{-1} (2)\).

Then \(\csc y = 2\), which means \(\sin y = \frac{1}{2}\).

The principal value branch of \(\csc^{-1} x\) is \([-\frac{\pi}{2}, \frac{\pi}{2}] \setminus \{0\}\).

We know that \(\sin \left( rac{\pi}{6}\right) = \frac{1}{2}\).

Since \(\frac{\pi}{6} \in [-\frac{\pi}{2}, \frac{\pi}{2}] \setminus \{0\}\), the principal value is \(\frac{\pi}{6}\).

$$ \csc^{-1} (2) = \frac{\pi}{6} $$

4. \(\tan^{-1} (-\sqrt{3})\)

Let \(y = \tan^{-1} (-\sqrt{3})\).

Then \(\tan y = -\sqrt{3}\).

The principal value branch of \(\tan^{-1} x\) is \((-\frac{\pi}{2}, \frac{\pi}{2})\).

We know that \(\tan \left( rac{\pi}{3}\right) = \sqrt{3}\).

Therefore, \(\tan \left(-\frac{\pi}{3}\right) = -\sqrt{3}\).

Since \(-\frac{\pi}{3} \in (-\frac{\pi}{2}, \frac{\pi}{2})\), the principal value is \(-\frac{\pi}{3}\).

$$ \tan^{-1} (-\sqrt{3}) = -\frac{\pi}{3} $$

5. \(\cos^{-1} \left(-\frac{1}{2}\right)\)

Let \(y = \cos^{-1} \left(-\frac{1}{2}\right)\).

Then \(\cos y = -\frac{1}{2}\).

The principal value branch of \(\cos^{-1} x\) is \([0, \pi]\).

We know that \(\cos \left( rac{\pi}{3}\right) = \frac{1}{2}\).

Since \(\cos (\pi - \theta) = -\cos \theta\), we have \(\cos \left(\pi - \frac{\pi}{3}\right) = -\cos \frac{\pi}{3} = -\frac{1}{2}\).

So, \(\cos \left(\frac{2\pi}{3}\right) = -\frac{1}{2}\).

Since \(\frac{2\pi}{3} \in [0, \pi]\), the principal value is \(\frac{2\pi}{3}\).

$$ \cos^{-1} \left(-\frac{1}{2}\right) = \frac{2\pi}{3} $$

6. \(\tan^{-1} (-1)\)

Let \(y = \tan^{-1} (-1)\).

Then \(\tan y = -1\).

The principal value branch of \(\tan^{-1} x\) is \((-\frac{\pi}{2}, \frac{\pi}{2})\).

We know that \(\tan \left( rac{\pi}{4}\right) = 1\).

Therefore, \(\tan \left(-\frac{\pi}{4}\right) = -1\).

Since \(-\frac{\pi}{4} \in (-\frac{\pi}{2}, \frac{\pi}{2})\), the principal value is \(-\frac{\pi}{4}\).

$$ \tan^{-1} (-1) = -\frac{\pi}{4} $$

7. \(\sec^{-1} \left(\frac{2}{\sqrt{3}}\right)\)

Let \(y = \sec^{-1} \left(\frac{2}{\sqrt{3}}\right)\).

Then \(\sec y = \frac{2}{\sqrt{3}}\), which means \(\cos y = \frac{\sqrt{3}}{2}\).

The principal value branch of \(\sec^{-1} x\) is \([0, \pi] \setminus \{\frac{\pi}{2}\}\).\)

We know that \(\cos \left( rac{\pi}{6}\right) = \frac{\sqrt{3}}{2}\).

Since \(\frac{\pi}{6} \in [0, \pi] \setminus \{\frac{\pi}{2}\}\), the principal value is \(\frac{\pi}{6}\).

$$ \sec^{-1} \left(\frac{2}{\sqrt{3}}\right) = \frac{\pi}{6} $$

8. \(\cot^{-1} (\sqrt{3})\)

Let \(y = \cot^{-1} (\sqrt{3})\).

Then \(\cot y = \sqrt{3}\).

The principal value branch of \(\cot^{-1} x\) is \((0, \pi)\).

We know that \(\cot \left( rac{\pi}{6}\right) = \sqrt{3}\).

Since \(\frac{\pi}{6} \in (0, \pi)\), the principal value is \(\frac{\pi}{6}\).

$$ \cot^{-1} (\sqrt{3}) = \frac{\pi}{6} $$

9. \(\cos^{-1} \left(-\frac{1}{\sqrt{2}}\right)\)

Let \(y = \cos^{-1} \left(-\frac{1}{\sqrt{2}}\right)\).

Then \(\cos y = -\frac{1}{\sqrt{2}}\).

The principal value branch of \(\cos^{-1} x\) is \([0, \pi]\).

We know that \(\cos \left( rac{\pi}{4}\right) = \frac{1}{\sqrt{2}}\).

Since \(\cos (\pi - \theta) = -\cos \theta\), we have \(\cos \left(\pi - \frac{\pi}{4}\right) = -\cos \frac{\pi}{4} = -\frac{1}{\sqrt{2}}\).

So, \(\cos \left(\frac{3\pi}{4}\right) = -\frac{1}{\sqrt{2}}\).

Since \(\frac{3\pi}{4} \in [0, \pi]\), the principal value is \(\frac{3\pi}{4}\).

$$ \cos^{-1} \left(-\frac{1}{\sqrt{2}}\right) = \frac{3\pi}{4} $$

10. \(\csc^{-1} (-\sqrt{2})\)

Let \(y = \csc^{-1} (-\sqrt{2})\).

Then \(\csc y = -\sqrt{2}\), which means \(\sin y = -\frac{1}{\sqrt{2}}\).

The principal value branch of \(\csc^{-1} x\) is \([-\frac{\pi}{2}, \frac{\pi}{2}] \setminus \{0\}\).

We know that \(\sin \left( rac{\pi}{4}\right) = \frac{1}{\sqrt{2}}\).

Therefore, \(\sin \left(-\frac{\pi}{4}\right) = -\frac{1}{\sqrt{2}}\).

Since \(-\frac{\pi}{4} \in [-\frac{\pi}{2}, \frac{\pi}{2}] \setminus \{0\}\), the principal value is \(-\frac{\pi}{4}\).

$$ \csc^{-1} (-\sqrt{2}) = -\frac{\pi}{4} $$

Find the values of the following:

11. \(\tan^{-1}(1) + \cos^{-1}\left(-\frac{1}{2}\right) + \sin^{-1}\left(-\frac{1}{2}\right)\)

We evaluate each term separately:

\(\tan^{-1}(1)\): Let \(y_1 = \tan^{-1}(1)\). \(\tan y_1 = 1\). Principal value is \(y_1 = \frac{\pi}{4}\).

\(\cos^{-1}\left(-\frac{1}{2}\right)\): Let \(y_2 = \cos^{-1}\left(-\frac{1}{2}\right)\). \(\cos y_2 = -\frac{1}{2}\). Principal value is \(y_2 = \frac{2\pi}{3}\) (from Q5).

\(\sin^{-1}\left(-\frac{1}{2}\right)\): Let \(y_3 = \sin^{-1}\left(-\frac{1}{2}\right)\). \(\sin y_3 = -\frac{1}{2}\). Principal value is \(y_3 = -\frac{\pi}{6}\) (from Q1).

Summing them:

$$ \frac{\pi}{4} + \frac{2\pi}{3} + \left(-\frac{\pi}{6}\right) = \frac{\pi}{4} + \frac{2\pi}{3} - \frac{\pi}{6} $$ Find a common denominator (12): $$ = \frac{3\pi}{12} + \frac{8\pi}{12} - \frac{2\pi}{12} = \frac{3\pi + 8\pi - 2\pi}{12} = \frac{9\pi}{12} = \frac{3\pi}{4} $$ $$ \tan^{-1}(1) + \cos^{-1}\left(-\frac{1}{2}\right) + \sin^{-1}\left(-\frac{1}{2}\right) = \frac{3\pi}{4} $$

12. \(\cos^{-1}\left(\frac{1}{2}\right) + 2\sin^{-1}\left(\frac{1}{2}\right)\)

We evaluate each term:

\(\cos^{-1}\left(\frac{1}{2}\right)\): Let \(y_1 = \cos^{-1}\left(\frac{1}{2}\right)\). \(\cos y_1 = \frac{1}{2}\). Principal value is \(y_1 = \frac{\pi}{3}\).

\(\sin^{-1}\left(\frac{1}{2}\right)\): Let \(y_2 = \sin^{-1}\left(\frac{1}{2}\right)\). \(\sin y_2 = \frac{1}{2}\). Principal value is \(y_2 = \frac{\pi}{6}\).

Summing them:

$$ \frac{\pi}{3} + 2\left(\frac{\pi}{6}\right) = \frac{\pi}{3} + \frac{\pi}{3} = \frac{2\pi}{3} $$ $$ \cos^{-1}\left(\frac{1}{2}\right) + 2\sin^{-1}\left(\frac{1}{2}\right) = \frac{2\pi}{3} $$

13. If \(\sin^{-1} x = y\), then

(A) \(0 \leq y \leq \pi\) (B) \(-\frac{\pi}{2} \leq y \leq \frac{\pi}{2}\) (C) \(0 < y < \pi\) (D) \(-\frac{\pi}{2} < y < \frac{\pi}{2}\)

The principal value branch of \(\sin^{-1} x\) is \([-\frac{\pi}{2}, \frac{\pi}{2}]\).

So, if \(y = \sin^{-1} x\), then \(y\) must be in this interval.

The correct option is (B).

14. \(\tan^{-1} \sqrt{3} - \sec^{-1} (-2)\) is equal to

(A) \(\pi\) (B) \(-\frac{\pi}{3}\) (C) \(\frac{\pi}{3}\) (D) \(\frac{2\pi}{3}\)

We evaluate each term:

\(\tan^{-1} \sqrt{3}\): Let \(y_1 = \tan^{-1} \sqrt{3}\). \(\tan y_1 = \sqrt{3}\). Principal value is \(y_1 = \frac{\pi}{3}\).

\(\sec^{-1} (-2)\): Let \(y_2 = \sec^{-1} (-2)\). \(\sec y_2 = -2\), so \(\cos y_2 = -\frac{1}{2}\).

The principal value branch of \(\sec^{-1} x\) is \([0, \pi] \setminus \{\frac{\pi}{2}\}\).

We know \(\cos (\pi - \frac{\pi}{3}) = -\cos \frac{\pi}{3} = -\frac{1}{2}\). So, \(\cos \frac{2\pi}{3} = -\frac{1}{2}\).

Since \(\frac{2\pi}{3} \in [0, \pi] \setminus \{\frac{\pi}{2}\}\), the principal value is \(y_2 = \frac{2\pi}{3}\).

Subtracting them:

$$ \frac{\pi}{3} - \frac{2\pi}{3} = -\frac{\pi}{3} $$

The correct option is (B).

Exercise 2.2

Prove the following:

1. \(3\sin^{-1} x = \sin^{-1} (3x - 4x^3)\), \(x \in \left[-\frac{1}{2}, \frac{1}{2}\right]\)

Let \(\sin^{-1} x = \theta\).

Then \(x = \sin \theta\).

Since \(x \in \left[-\frac{1}{2}, \frac{1}{2}\right]\), \(\theta \in \left[-\frac{\pi}{6}, \frac{\pi}{6}\right]\).

Consider the RHS: \(\sin^{-1} (3x - 4x^3)\).

Substitute \(x = \sin \theta\): $$ \sin^{-1} (3\sin\theta - 4\sin^3\theta) $$ We know the trigonometric identity \(3\sin\theta - 4\sin^3\theta = \sin 3\theta\).

$$ = \sin^{-1} (\sin 3\theta) $$ Since \(\theta \in \left[-\frac{\pi}{6}, \frac{\pi}{6}\right]\), it follows that \(3\theta \in \left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\).

In the interval \([-\frac{\pi}{2}, \frac{\pi}{2}]\), \(\sin^{-1} (\sin A) = A\).

So, \( \sin^{-1} (\sin 3\theta) = 3\theta \).

Substitute back \(\theta = \sin^{-1} x\): $$ = 3\sin^{-1} x $$ Thus, \(3\sin^{-1} x = \sin^{-1} (3x - 4x^3)\) is proven for the given domain.

2. \(3\cos^{-1} x = \cos^{-1} (4x^3 - 3x)\), \(x \in \left[\frac{1}{2}, 1\right]\)

Let \(\cos^{-1} x = \theta\).

Then \(x = \cos \theta\).

Since \(x \in \left[\frac{1}{2}, 1\right]\), \(\theta \in \left[0, \frac{\pi}{3}\right]\).

Consider the RHS: \(\cos^{-1} (4x^3 - 3x)\).

Substitute \(x = \cos \theta\): $$ \cos^{-1} (4\cos^3\theta - 3\cos\theta) $$ We know the trigonometric identity \(4\cos^3\theta - 3\cos\theta = \cos 3\theta\).

$$ = \cos^{-1} (\cos 3\theta) $$ Since \(\theta \in \left[0, \frac{\pi}{3}\right]\), it follows that \(3\theta \in [0, \pi]\).

In the interval \([0, \pi]\), \(\cos^{-1} (\cos A) = A\).

So, \( \cos^{-1} (\cos 3\theta) = 3\theta \).

Substitute back \(\theta = \cos^{-1} x\): $$ = 3\cos^{-1} x $$ Thus, \(3\cos^{-1} x = \cos^{-1} (4x^3 - 3x)\) is proven for the given domain.

3. \(\tan^{-1} \left(\frac{\sqrt{1+x^2} - 1}{x}\right)\), \(x \neq 0\) in the simplest form.

Let \(x = \tan \theta\).

Then \(\theta = \tan^{-1} x\).

Substitute \(x = \tan \theta\) into the expression: $$ \tan^{-1} \left(\frac{\sqrt{1+\tan^2\theta} - 1}{\tan\theta}\right) $$ We know \(\sqrt{1+\tan^2\theta} = \sqrt{\sec^2\theta} = |\sec\theta|\).

Assuming \(\theta \in (-\frac{\pi}{2}, \frac{\pi}{2})\) (principal value branch of \(\tan^{-1} x\)), \(\sec\theta > 0\), so \(|\sec\theta| = \sec\theta\).

$$ = \tan^{-1} \left(\frac{\sec\theta - 1}{\tan\theta}\right) $$ Convert to \(\sin\) and \(\cos\): $$ = \tan^{-1} \left(\frac{\frac{1}{\cos\theta} - 1}{\frac{\sin\theta}{\cos\theta}}\right) = \tan^{-1} \left(\frac{\frac{1-\cos\theta}{\cos\theta}}{\frac{\sin\theta}{\cos\theta}}\right) = \tan^{-1} \left(\frac{1-\cos\theta}{\sin\theta}\right) $$ Use half-angle formulas: \(1-\cos\theta = 2\sin^2\left(\frac{\theta}{2}\right)\) and \(\sin\theta = 2\sin\left(\frac{\theta}{2}\right)\cos\left(\frac{\theta}{2}\right)\).

$$ = \tan^{-1} \left(\frac{2\sin^2\left(\frac{\theta}{2}\right)}{2\sin\left(\frac{\theta}{2}\right)\cos\left(\frac{\theta}{2}\right)}\right) = \tan^{-1} \left(\frac{\sin\left(\frac{\theta}{2}\right)}{\cos\left(\frac{\theta}{2}\right)}\right) = \tan^{-1} \left(\tan\left(\frac{\theta}{2}\right)\right) $$ Since \(\theta \in (-\frac{\pi}{2}, \frac{\pi}{2})\), it implies \(\frac{\theta}{2} \in (-\frac{\pi}{4}, \frac{\pi}{4})\). In this interval, \(\tan^{-1}(\tan A) = A\).

$$ = \frac{\theta}{2} $$ Substitute back \(\theta = \tan^{-1} x\): $$ = \frac{1}{2}\tan^{-1} x $$ The simplest form is \(\frac{1}{2}\tan^{-1} x\).

4. \(\tan^{-1} \left(\sqrt{\frac{1-\cos x}{1+\cos x}}\right)\), \(0 < x < \pi\) in the simplest form.

Given the expression \(\tan^{-1} \left(\sqrt{\frac{1-\cos x}{1+\cos x}}\right)\).

Use half-angle formulas: \(1-\cos x = 2\sin^2\left(\frac{x}{2}\right)\) and \(1+\cos x = 2\cos^2\left(\frac{x}{2}\right)\).

$$ \sqrt{\frac{1-\cos x}{1+\cos x}} = \sqrt{\frac{2\sin^2\left(\frac{x}{2}\right)}{2\cos^2\left(\frac{x}{2}\right)}} = \sqrt{\tan^2\left(\frac{x}{2}\right)} = \left|\tan\left(\frac{x}{2}\right)\right| $$ Given \(0 < x < \pi\), this implies \(0 < \frac{x}{2} < \frac{\pi}{2}\).

In this interval, \(\tan\left(\frac{x}{2}\right) > 0\), so \(\left|\tan\left(\frac{x}{2}\right)\right| = \tan\left(\frac{x}{2}\right)\).

So the expression becomes: $$ \tan^{-1} \left(\tan\left(\frac{x}{2}\right)\right) $$ Since \(0 < \frac{x}{2} < \frac{\pi}{2}\), which is within the principal value branch of \(\tan^{-1} A\) (\((-\frac{\pi}{2}, \frac{\pi}{2})\)), \(\tan^{-1}(\tan A) = A\).

$$ = \frac{x}{2} $$ The simplest form is \(\frac{x}{2}\).

5. \(\tan^{-1} \left(\frac{\cos x - \sin x}{\cos x + \sin x}\right)\), \(-\frac{\pi}{4} < x < \frac{3\pi}{4}\) in the simplest form.

Given the expression \(\tan^{-1} \left(\frac{\cos x - \sin x}{\cos x + \sin x}\right)\).

Divide the numerator and denominator by \(\cos x\):

$$ \tan^{-1} \left(\frac{1 - \frac{\sin x}{\cos x}}{1 + \frac{\sin x}{\cos x}}\right) = \tan^{-1} \left(\frac{1 - \tan x}{1 + \tan x}\right) $$ We know the tangent addition formula: \(\tan(A-B) = \frac{\tan A - \tan B}{1 + \tan A \tan B}\).

Comparing, we can write \(1\) as \(\tan\left(\frac{\pi}{4}\right)\):

$$ = \tan^{-1} \left(\frac{\tan\left(\frac{\pi}{4}\right) - \tan x}{1 + \tan\left(\frac{\pi}{4}\right)\tan x}\right) = \tan^{-1} \left(\tan\left(\frac{\pi}{4} - x\right)\right) $$ Given \(-\frac{\pi}{4} < x < \frac{3\pi}{4}\).

Multiply by \(-1\): \(-\frac{3\pi}{4} < -x < \frac{\pi}{4}\). Add \(\frac{\pi}{4}\): \(-\frac{3\pi}{4} + \frac{\pi}{4} < \frac{\pi}{4} - x < \frac{\pi}{4} + \frac{\pi}{4}\). So, \(-\frac{2\pi}{4} < \frac{\pi}{4} - x < \frac{2\pi}{4}\), which simplifies to \(-\frac{\pi}{2} < \frac{\pi}{4} - x < \frac{\pi}{2}\).

This interval \((-\frac{\pi}{2}, \frac{\pi}{2})\) is the principal value branch of \(\tan^{-1} A\).

So, \( \tan^{-1} \left(\tan\left(\frac{\pi}{4} - x\right)\right) = \frac{\pi}{4} - x \). The simplest form is \(\frac{\pi}{4} - x\).

6. \(\tan^{-1} \left(\frac{x}{\sqrt{a^2 - x^2}}\right)\), \(|x| < a\) in the simplest form.

Given the expression \(\tan^{-1} \left(\frac{x}{\sqrt{a^2 - x^2}}\right)\).

Since \(|x| < a\), we can substitute \(x = a \sin \theta\). Then \(\theta = \sin^{-1}\left(\frac{x}{a}\right)\). The range for \(\theta\) is \((-\frac{\pi}{2}, \frac{\pi}{2})\).

Substitute \(x = a \sin \theta\): $$ \tan^{-1} \left(\frac{a \sin \theta}{\sqrt{a^2 - (a \sin \theta)^2}}\right) = \tan^{-1} \left(\frac{a \sin \theta}{\sqrt{a^2 - a^2 \sin^2\theta}}\right) $$ $$ = \tan^{-1} \left(\frac{a \sin \theta}{\sqrt{a^2(1 - \sin^2\theta)}}\right) = \tan^{-1} \left(\frac{a \sin \theta}{\sqrt{a^2\cos^2\theta}}\right) $$ $$ = \tan^{-1} \left(\frac{a \sin \theta}{|a \cos \theta|}\right) $$ Since \(a > 0\) (implied by \(\sqrt{a^2}\)) and \(\theta \in (-\frac{\pi}{2}, \frac{\pi}{2})\), \(\cos\theta > 0\), so \(|a \cos \theta| = a \cos \theta\).

$$ = \tan^{-1} \left(\frac{a \sin \theta}{a \cos \theta}\right) = \tan^{-1} (\tan \theta) $$ Since \(\theta \in (-\frac{\pi}{2}, \frac{\pi}{2})\), \(\tan^{-1}(\tan \theta) = \theta\).

Substitute back \(\theta = \sin^{-1}\left(\frac{x}{a}\right)\): $$ = \sin^{-1}\left(\frac{x}{a}\right) $$ The simplest form is \(\sin^{-1}\left(\frac{x}{a}\right)\).

7. \(\tan^{-1} \left(\frac{3a^2x - x^3}{a^3 - 3ax^2}\right)\), \(a > 0; -\frac{a}{\sqrt{3}} < x < \frac{a}{\sqrt{3}}\) in the simplest form.

Given the expression \(\tan^{-1} \left(\frac{3a^2x - x^3}{a^3 - 3ax^2}\right)\).

To match the \(\tan 3\theta\) formula, we should substitute \(x = a \tan \theta\). Then \(\theta = \tan^{-1}\left(\frac{x}{a}\right)\). Given \(-\frac{a}{\sqrt{3}} < x < \frac{a}{\sqrt{3}}\). Dividing by \(a\): \(-\frac{1}{\sqrt{3}} < \frac{x}{a} < \frac{1}{\sqrt{3}}\). So, \(-\frac{1}{\sqrt{3}} < \tan \theta < \frac{1}{\sqrt{3}}\), which means \(-\frac{\pi}{6} < \theta < \frac{\pi}{6}\). Substitute \(x = a \tan \theta\) into the expression: $$ \tan^{-1} \left(\frac{3a^2(a \tan \theta) - (a \tan \theta)^3}{a^3 - 3a(a \tan \theta)^2}\right) = \tan^{-1} \left(\frac{3a^3 \tan \theta - a^3 \tan^3\theta}{a^3 - 3a^3 \tan^2\theta}\right) $$ Factor out \(a^3\) from numerator and denominator: $$ = \tan^{-1} \left(\frac{a^3(3 \tan \theta - \tan^3\theta)}{a^3(1 - 3 \tan^2\theta)}\right) = \tan^{-1} \left(\frac{3 \tan \theta - \tan^3\theta}{1 - 3 \tan^2\theta}\right) $$ We know the trigonometric identity \(\frac{3 \tan \theta - \tan^3\theta}{1 - 3 \tan^2\theta} = \tan 3\theta\). $$ = \tan^{-1} (\tan 3\theta) $$ Since \(-\frac{\pi}{6} < \theta < \frac{\pi}{6}\), it follows that \(-\frac{\pi}{2} < 3\theta < \frac{\pi}{2}\). This interval \((-\frac{\pi}{2}, \frac{\pi}{2})\) is the principal value branch of \(\tan^{-1} A\). So, \( \tan^{-1} (\tan 3\theta) = 3\theta \). Substitute back \(\theta = \tan^{-1}\left(\frac{x}{a}\right)\): $$ = 3\tan^{-1}\left(\frac{x}{a}\right) $$ The simplest form is \(3\tan^{-1}\left(\frac{x}{a}\right)\).

Next, evaluate \(\cos\left(2\sin^{-1}\left(\frac{1}{2}\right)\right)\): \(\cos\left(\frac{\pi}{3}\right) = \frac{1}{2}\).

Finally, evaluate \(\tan^{-1} \left[2\cos\left(2\sin^{-1}\left(\frac{1}{2}\right)\right)\right]\): \(\tan^{-1} \left[2\left(\frac{1}{2}\right)\right] = \tan^{-1} (1)\).

9. \(\tan \frac{1}{2}\left[\sin^{-1} \left(\frac{2x}{1+x^2}\right) + \cos^{-1} \left(\frac{1-y^2}{1+y^2}\right)\right]\), \(|x| < 1, y > 0\) and \(xy < 1\).

We use standard substitutions for the inverse trigonometric terms:

For \(\sin^{-1} \left(\frac{2x}{1+x^2}\right)\): Let \(x = \tan A\). Then \(A = \tan^{-1} x\).

Since \(|x| < 1\), \(-\frac{\pi}{4} < A < \frac{\pi}{4}\). So \(-\frac{\pi}{2} < 2A < \frac{\pi}{2}\). \(\sin^{-1} \left(\frac{2\tan A}{1+\tan^2 A}\right) = \sin^{-1} (\sin 2A) = 2A = 2\tan^{-1} x\).

For \(\cos^{-1} \left(\frac{1-y^2}{1+y^2}\right)\): Let \(y = \tan B\). Then \(B = \tan^{-1} y\).

Since \(y > 0\), \(0 < B < \frac{\pi}{2}\). So \(0 < 2B < \pi\). \(\cos^{-1} \left(\frac{1-\tan^2 B}{1+\tan^2 B}\right) = \cos^{-1} (\cos 2B) = 2B = 2\tan^{-1} y\).

Substitute these back into the expression:

$$ \tan \frac{1}{2} \left[2\tan^{-1} x + 2\tan^{-1} y\right] $$ $$ = \tan \frac{1}{2} \left[2(\tan^{-1} x + \tan^{-1} y)\right] $$ $$ = \tan (\tan^{-1} x + \tan^{-1} y) $$ We use the formula \(\tan^{-1} x + \tan^{-1} y = \tan^{-1} \left(\frac{x+y}{1-xy}\right)\) when \(xy < 1\). The condition \(xy < 1\) is given in the problem.

$$ = \tan \left(\tan^{-1} \left(\frac{x+y}{1-xy}\right)\right) $$ $$ = \frac{x+y}{1-xy} $$ The value of the expression is \(\frac{x+y}{1-xy}\).

10. \(\sin\left(\sin^{-1}\left(\frac{2\pi}{3}\right)\right)\) (Typo in problem, should be \( \sin\left(\sin^{-1}(\frac{2}{3})\right) \) or \( \sin^{-1}(\sin(\frac{2\pi}{3})) \). Assuming the latter interpretation)

Assuming the question is \(\sin^{-1}\left(\sin\left(\frac{2\pi}{3}\right)\right)\):

We need to find the principal value of \(\sin^{-1} (\sin \theta)\). The principal value branch of \(\sin^{-1} x\) is \([-\frac{\pi}{2}, \frac{\pi}{2}]\).

\(\sin\left(\frac{2\pi}{3}\right) = \sin\left(\pi - \frac{\pi}{3}\right) = \sin\left(\frac{\pi}{3}\right) = \frac{\sqrt{3}}{2}\).

So the expression becomes \(\sin^{-1}\left(\frac{\sqrt{3}}{2}\right)\). We know \(\sin\left(\frac{\pi}{3}\right) = \frac{\sqrt{3}}{2}\). Since \(\frac{\pi}{3} \in [-\frac{\pi}{2}, \frac{\pi}{2}]\), the principal value is \(\frac{\pi}{3}\).

$$ \sin^{-1}\left(\sin\left(\frac{2\pi}{3}\right)\right) = \frac{\pi}{3} $$

If the question was literally \(\sin\left(\sin^{-1}\left(\frac{2\pi}{3}\right)\right)\): The domain of \(\sin^{-1} x\) is \([-1, 1]\). Since \(\frac{2\pi}{3} \approx 2.09\), it is outside the domain \([-1, 1]\). So, \(\sin^{-1}\left(\frac{2\pi}{3}\right)\) is not defined for real numbers. This would make the entire expression undefined.

Based on typical textbook problems, the interpretation \(\sin^{-1}(\sin \theta)\) is more likely.

14. \(\sin^{-1} (1-x) - 2\sin^{-1} x = \frac{\pi}{2}\), then \(x\) is equal to

(A) \(0, \frac{1}{2}\) (B) \(1, \frac{1}{2}\) (C) \(0\) (D) \(\frac{1}{2}\)

Given the equation: \(\sin^{-1} (1-x) - 2\sin^{-1} x = \frac{\pi}{2}\). Let \(\sin^{-1} x = A\). Then \(x = \sin A\). The domain of \(\sin^{-1} x\) is \([-1, 1]\). So \(x \in [-1, 1]\). Also, \(1-x \in [-1, 1]\), so \(0 \leq x \leq 2\). Combining both, \(x \in [0, 1]\). Rearrange the equation: $$ \sin^{-1} (1-x) = \frac{\pi}{2} + 2\sin^{-1} x $$ Take \(\sin\) of both sides: $$ 1-x = \sin\left(\frac{\pi}{2} + 2\sin^{-1} x\right) $$ We know \(\sin(\frac{\pi}{2} + \theta) = \cos \theta\). $$ 1-x = \cos(2\sin^{-1} x) $$ Let \(\sin^{-1} x = A\). Then \(1-x = \cos(2A)\). We know \(\cos(2A) = 1 - 2\sin^2 A\). Since \(\sin A = x\), we have \(\cos(2A) = 1 - 2x^2\). Substitute this back: $$ 1-x = 1 - 2x^2 $$ $$ -x = -2x^2 $$ $$ 2x^2 - x = 0 $$ Factor out \(x\): $$ x(2x - 1) = 0 $$ This gives two possible solutions: \(x = 0\) or \(x = \frac{1}{2}\).

Now, we must check these solutions in the original equation and principal value ranges.

Check \(x = 0\): LHS: \(\sin^{-1} (1-0) - 2\sin^{-1} (0) = \sin^{-1}(1) - 2(0) = \frac{\pi}{2} - 0 = \frac{\pi}{2}\). RHS: \(\frac{\pi}{2}\). LHS = RHS. So, \(x = 0\) is a solution.

Check \(x = \frac{1}{2}\): LHS: \(\sin^{-1} (1-\frac{1}{2}) - 2\sin^{-1} (\frac{1}{2}) = \sin^{-1}(\frac{1}{2}) - 2\sin^{-1}(\frac{1}{2})\). Since \(\sin^{-1}(\frac{1}{2}) = \frac{\pi}{6}\). LHS: \( \frac{\pi}{6} - 2\left(\frac{\pi}{6}\right) = \frac{\pi}{6} - \frac{2\pi}{6} = -\frac{\pi}{6}\). RHS: \(\frac{\pi}{2}\). LHS \(\neq\) RHS. So, \(x = \frac{1}{2}\) is NOT a solution.

Therefore, the only solution is \(x = 0\).

The correct option is (C).
