1. Find an anti-derivative (or integral) of the following function by the method of inspection: \(\sin 2x\)

We need to find a function whose derivative is \(\sin 2x\).

We know that \( \frac{d}{dx}(\cos 2x) = -2 \sin 2x \).

So, \( \sin 2x = -\frac{1}{2} \frac{d}{dx}(\cos 2x) = \frac{d}{dx}\left(-\frac{1}{2}\cos 2x\right) \).

Therefore, an anti-derivative of \(\sin 2x\) is \(-\frac{1}{2}\cos 2x\).

2. Find an anti-derivative of: \(\cos 3x\)

We need to find a function whose derivative is \(\cos 3x\).

We know that \( \frac{d}{dx}(\sin 3x) = 3 \cos 3x \).

So, \( \cos 3x = \frac{1}{3} \frac{d}{dx}(\sin 3x) = \frac{d}{dx}\left(\frac{1}{3}\sin 3x\right) \).

Therefore, an anti-derivative of \(\cos 3x\) is \(\frac{1}{3}\sin 3x\).

3. Find an anti-derivative of: \(e^{2x}\)

We need to find a function whose derivative is \(e^{2x}\).

We know that \( \frac{d}{dx}(e^{2x}) = 2e^{2x} \).

So, \( e^{2x} = \frac{1}{2} \frac{d}{dx}(e^{2x}) = \frac{d}{dx}\left(\frac{1}{2}e^{2x}\right) \).

Therefore, an anti-derivative of \(e^{2x}\) is \(\frac{1}{2}e^{2x}\).

4. Find an anti-derivative of: \((ax + b)^2\)

We need to find a function whose derivative is \((ax+b)^2\).

We know that \( \frac{d}{dx}((ax+b)^3) = 3(ax+b)^2 \cdot a = 3a(ax+b)^2 \).

So, \( (ax+b)^2 = \frac{1}{3a} \frac{d}{dx}((ax+b)^3) = \frac{d}{dx}\left(\frac{(ax+b)^3}{3a}\right) \).

Therefore, an anti-derivative of \((ax+b)^2\) is \(\frac{(ax+b)^3}{3a}\).

5. Find an anti-derivative of: \(\sin 2x - 4e^{3x}\)

We find the anti-derivative of each term separately.

Anti-derivative of \(\sin 2x\) is \(-\frac{1}{2}\cos 2x\) (from Q1).

For \(4e^{3x}\), we know \( \frac{d}{dx}(e^{3x}) = 3e^{3x} \), so \( \frac{d}{dx}\left(\frac{4}{3}e^{3x}\right) = 4e^{3x} \).

Thus, an anti-derivative of \(\sin 2x - 4e^{3x}\) is \(-\frac{1}{2}\cos 2x - \frac{4}{3}e^{3x}\).

1. \(\frac{2x}{1+x^2}\)

Let \(I = \int \frac{2x}{1+x^2} dx\). Let \(u = 1+x^2\), then \(du = 2x dx\).

$$ I = \int \frac{1}{u} du = \log|u| + C = \log(1+x^2) + C $$

(Since \(1+x^2\) is always positive, absolute value is not needed).

2. \(\frac{(\log x)^2}{x}\)

Let \(I = \int \frac{(\log x)^2}{x} dx\). Let \(u = \log x\), then \(du = \frac{1}{x} dx\).

$$ I = \int u^2 du = \frac{u^3}{3} + C = \frac{(\log x)^3}{3} + C $$

3. \(\frac{1}{x+x\log x}\)

Let \(I = \int \frac{1}{x+x\log x} dx = \int \frac{1}{x(1+\log x)} dx\).

Let \(u = 1+\log x\), then \(du = \frac{1}{x} dx\).

$$ I = \int \frac{1}{u} du = \log|u| + C = \log|1+\log x| + C $$

4. \(\sin x \sin(\cos x)\)

Let \(I = \int \sin x \sin(\cos x) dx\). Let \(u = \cos x\), then \(du = -\sin x dx\).

$$ I = \int -\sin(u) du = \cos(u) + C = \cos(\cos x) + C $$

5. \(\sin(ax+b)\cos(ax+b)\)

Let \(I = \int \sin(ax+b)\cos(ax+b) dx\). Let \(u = \sin(ax+b)\), then \(du = a\cos(ax+b) dx\).

$$ I = \frac{1}{a} \int u du = \frac{1}{a} \frac{u^2}{2} + C = \frac{\sin^2(ax+b)}{2a} + C $$

Alternatively, \(\sin(ax+b)\cos(ax+b) = \frac{1}{2}\sin(2(ax+b))\).

$$ I = \int \frac{1}{2}\sin(2ax+2b) dx = \frac{1}{2} \left( -\frac{\cos(2ax+2b)}{2a} \right) + C = -\frac{\cos(2(ax+b))}{4a} + C $$

6. \(\sqrt{ax+b}\)

Let \(I = \int \sqrt{ax+b} dx\). Let \(u=ax+b\), then \(du=a dx\).

$$ I = \frac{1}{a} \int \sqrt{u} du = \frac{1}{a} \int u^{1/2} du = \frac{1}{a} \frac{u^{3/2}}{3/2} + C = \frac{2}{3a}(ax+b)^{3/2} + C $$

7. \(x\sqrt{x+2}\)

Let \(I = \int x\sqrt{x+2} dx\). Let \(u=x+2\), then \(du=dx\) and \(x=u-2\).

$$ I = \int (u-2)\sqrt{u} du = \int (u^{3/2} - 2u^{1/2}) du $$ $$ = \frac{u^{5/2}}{5/2} - 2\frac{u^{3/2}}{3/2} + C = \frac{2}{5}(x+2)^{5/2} - \frac{4}{3}(x+2)^{3/2} + C $$
