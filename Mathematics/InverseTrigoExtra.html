Inverse Trigonometry Worksheet Solutions

Two Marks Questions: LEVEL - 1

1. Evaluate: \(\sin^{-1}(\sin(\frac{33\pi}{7})) + \cot^{-1}(\cot(-\frac{19\pi}{8}))\)

We evaluate each term separately. For \(\sin^{-1}(\sin(\frac{33\pi}{7}))\), we find an equivalent angle in the principal range \([-\pi/2, \pi/2]\). \(\frac{33\pi}{7} = 4\pi + \frac{5\pi}{7}\), so \(\sin(\frac{33\pi}{7}) = \sin(\frac{5\pi}{7}) = \sin(\pi - \frac{2\pi}{7}) = \sin(\frac{2\pi}{7})\). Thus, \(\sin^{-1}(\sin(\frac{33\pi}{7})) = \frac{2\pi}{7}\). For \(\cot^{-1}(\cot(-\frac{19\pi}{8}))\), the principal range is \((0, \pi)\). Using \(\cot(-\theta) = -\cot\theta\) and \(\cot^{-1}(-x) = \pi - \cot^{-1}x\), the term is \(\pi - \cot^{-1}(\cot(\frac{19\pi}{8}))\). \(\frac{19\pi}{8} = 2\pi + \frac{3\pi}{8}\), so \(\cot(\frac{19\pi}{8}) = \cot(\frac{3\pi}{8})\). The term becomes \(\pi - \cot^{-1}(\cot(\frac{3\pi}{8})) = \pi - \frac{3\pi}{8} = \frac{5\pi}{8}\). The sum is \(\frac{2\pi}{7} + \frac{5\pi}{8} = \frac{16\pi + 35\pi}{56} = \frac{51\pi}{56}\).

2. Evaluate: \(\cos^{-1}(\cos(-\frac{33\pi}{7})) + \tan^{-1}(\tan(\frac{13\pi}{8}))\)

For the first term, \(\cos^{-1}(\cos(-\frac{33\pi}{7}))\), the principal range is \([0, \pi]\). Using \(\cos(-\theta) = \cos\theta\), we have \(\cos^{-1}(\cos(\frac{33\pi}{7}))\). Since \(\frac{33\pi}{7} = 4\pi + \frac{5\pi}{7}\), \(\cos(\frac{33\pi}{7}) = \cos(\frac{5\pi}{7})\). As \(\frac{5\pi}{7} \in [0, \pi]\), the value is \(\frac{5\pi}{7}\). For the second term, \(\tan^{-1}(\tan(\frac{13\pi}{8}))\), the principal range is \((-\pi/2, \pi/2)\). Since \(\frac{13\pi}{8} = \pi + \frac{5\pi}{8}\), \(\tan(\frac{13\pi}{8}) = \tan(\frac{5\pi}{8})\). To bring the angle into the range, we use \(\tan\theta = \tan(\theta-\pi)\): \(\tan(\frac{5\pi}{8}) = \tan(\frac{5\pi}{8}-\pi) = \tan(-\frac{3\pi}{8})\). The value is \(-\frac{3\pi}{8}\). The sum is \(\frac{5\pi}{7} - \frac{3\pi}{8} = \frac{40\pi - 21\pi}{56} = \frac{19\pi}{56}\).

3. Evaluate: \(\sin^{-1}(\cos(\frac{\pi}{9}))\)

To evaluate the expression, we convert the inner cosine term to a sine term using the identity \(\cos\theta = \sin(\frac{\pi}{2} - \theta)\). \[ \sin^{-1}\left(\cos\left(\frac{\pi}{9}\right)\right) = \sin^{-1}\left(\sin\left(\frac{\pi}{2} - \frac{\pi}{9}\right)\right) = \sin^{-1}\left(\sin\left(\frac{7\pi}{18}\right)\right) \] Since \(\frac{7\pi}{18}\) is in the principal value range of \(\sin^{-1}x\), \([-\frac{\pi}{2}, \frac{\pi}{2}]\), the answer is \(\frac{7\pi}{18}\).

4. Write a branch of \(\sec^{-1}x\) other than the principal branch.

The principal value branch for the range of \(y=\sec^{-1}x\) is \([0, \pi] \setminus \{\frac{\pi}{2}\}\). This corresponds to restricting the domain of \(\sec x\) to make it one-to-one. Any other interval where \(\sec x\) is one-to-one can define another branch. For example, the interval \([\pi, 2\pi]\) is a suitable choice. In this interval, \(\sec x\) is undefined at \(x = \frac{3\pi}{2}\). Therefore, another branch for the range of \(\sec^{-1}x\) is \([\pi, 2\pi] \setminus \{\frac{3\pi}{2}\}\).

5. Write the value of \(\tan^{-1}(2\sin(2\cos^{-1}(\frac{\sqrt{3}}{2})))\).

We evaluate the expression from the inside out. First, \(\cos^{-1}(\frac{\sqrt{3}}{2}) = \frac{\pi}{6}\). The expression becomes \(\tan^{-1}(2\sin(2 \cdot \frac{\pi}{6})) = \tan^{-1}(2\sin(\frac{\pi}{3}))\). Since \(\sin(\frac{\pi}{3}) = \frac{\sqrt{3}}{2}\), we have \(\tan^{-1}(2 \cdot \frac{\sqrt{3}}{2}) = \tan^{-1}(\sqrt{3})\). The principal value is \(\frac{\pi}{3}\).

6. Evaluate \(\cos[\cos^{-1}(-\frac{\sqrt{3}}{2}) + \frac{\pi}{6}]\)

First, we evaluate \(\cos^{-1}(-\frac{\sqrt{3}}{2})\). Using \(\cos^{-1}(-u) = \pi - \cos^{-1}u\), we get: \[\cos^{-1}(-\frac{\sqrt{3}}{2}) = \pi - \cos^{-1}(\frac{\sqrt{3}}{2}) = \pi - \frac{\pi}{6} = \frac{5\pi}{6}\] Substituting this into the expression: \[\cos\left(\frac{5\pi}{6} + \frac{\pi}{6}\right) = \cos\left(\frac{6\pi}{6}\right) = \cos(\pi) = -1\]

7. If \(\cos(\sin^{-1}\frac{2}{5} + \cos^{-1}x) = 0\), then \(x\) is equal to...

Given \(\cos(\sin^{-1}\frac{2}{5} + \cos^{-1}x) = 0\). For the cosine of an angle to be zero, the angle must be an odd multiple of \(\pi/2\). The range of \(\sin^{-1}y + \cos^{-1}x\) suggests the principal value \(\pi/2\). \[ \sin^{-1}\frac{2}{5} + \cos^{-1}x = \frac{\pi}{2} \] We use the identity \(\sin^{-1}y + \cos^{-1}y = \frac{\pi}{2}\). Comparing the two equations, we see that \(x\) must be equal to \(\frac{2}{5}\). Answer: (b) \(\frac{2}{5}\).

The domain of the function defined by \(f(x) = \sin^{-1}\sqrt{x-1}\) is...

For the function \(f(x) = \sin^{-1}\sqrt{x-1}\) to be defined, two conditions must be met: 1. The expression under the square root must be non-negative: \(x-1 \ge 0 \implies x \ge 1\). 2. The argument of \(\sin^{-1}\) must be in the interval \([-1, 1]\): \(-1 \le \sqrt{x-1} \le 1\). Since \(\sqrt{x-1}\) is always non-negative, this simplifies to \(0 \le \sqrt{x-1} \le 1\). Squaring all parts of the inequality gives \(0 \le x-1 \le 1\), which implies \(1 \le x \le 2\). Combining both conditions (\(x \ge 1\) and \(1 \le x \le 2\)), the domain is \([1, 2]\). Answer: (a) \([1, 2]\).

LEVEL - 2

1. Simplify: \(\sin^{-1}(\frac{\sin x + \cos x}{\sqrt{2}})\)

We can rewrite the argument of \(\sin^{-1}\) using the sine addition formula: \[ \frac{\sin x + \cos x}{\sqrt{2}} = \frac{1}{\sqrt{2}}\sin x + \frac{1}{\sqrt{2}}\cos x = \cos\left(\frac{\pi}{4}\right)\sin x + \sin\left(\frac{\pi}{4}\right)\cos x = \sin\left(x+\frac{\pi}{4}\right) \] The expression becomes \(\sin^{-1}(\sin(x+\frac{\pi}{4}))\). If \(x+\frac{\pi}{4}\) is in the principal range of \(\sin^{-1}\), i.e., \([-\pi/2, \pi/2]\), the simplified form is \(x+\frac{\pi}{4}\).

2. Find \(\tan[\frac{1}{2}\cos^{-1}\frac{3}{11}]\)

Let \(\theta = \frac{1}{2}\cos^{-1}\frac{3}{11}\). Then \(2\theta = \cos^{-1}\frac{3}{11}\), which means \(\cos(2\theta) = \frac{3}{11}\). We need to find \(\tan\theta\). We use the half-angle identity for tangent: \[ \tan^2\theta = \frac{1-\cos(2\theta)}{1+\cos(2\theta)} = \frac{1 - 3/11}{1 + 3/11} = \frac{8/11}{14/11} = \frac{8}{14} = \frac{4}{7} \] Since the range of \(\cos^{-1}\) is \([0, \pi]\), \(2\theta \in [0, \pi]\), so \(\theta \in [0, \pi/2]\). In this quadrant, \(\tan\theta\) is positive. \[ \tan\theta = \sqrt{\frac{4}{7}} = \frac{2}{\sqrt{7}} \]

3. If \(\sin^{-1}x > \cos^{-1}x\), then \(x\) belongs to which interval.

Given the inequality \(\sin^{-1}x > \cos^{-1}x\). Using the identity \(\cos^{-1}x = \frac{\pi}{2} - \sin^{-1}x\), we substitute it into the inequality: \[ \sin^{-1}x > \frac{\pi}{2} - \sin^{-1}x \] \[ 2\sin^{-1}x > \frac{\pi}{2} \implies \sin^{-1}x > \frac{\pi}{4} \] Since \(\sin\) is an increasing function on its principal domain, we can apply it to both sides: \[ \sin(\sin^{-1}x) > \sin\left(\frac{\pi}{4}\right) \implies x > \frac{1}{\sqrt{2}} \] The domain of \(\sin^{-1}x\) is \([-1, 1]\). Combining this with our result, the solution is \(x \in (\frac{1}{\sqrt{2}}, 1]\).

LEVEL - 3

1. If \(\sin^{-1}x - \cos^{-1}x = \frac{\pi}{6}\), find \(x\).

We are given the equation: \(\sin^{-1}x - \cos^{-1}x = \frac{\pi}{6} \quad \cdots(1)\) We also know the identity: \(\sin^{-1}x + \cos^{-1}x = \frac{\pi}{2} \quad \cdots(2)\) Adding equation (1) and (2) eliminates \(\cos^{-1}x\): \[ 2\sin^{-1}x = \frac{\pi}{6} + \frac{\pi}{2} = \frac{\pi + 3\pi}{6} = \frac{4\pi}{6} = \frac{2\pi}{3} \] \[ \sin^{-1}x = \frac{\pi}{3} \implies x = \sin\left(\frac{\pi}{3}\right) = \frac{\sqrt{3}}{2} \]

2. If \(\tan^{-1}4 + \cot^{-1}x = \frac{\pi}{2}\), find \(x\).

We use the complementary identity \(\tan^{-1}y + \cot^{-1}y = \frac{\pi}{2}\). Given the equation \(\tan^{-1}4 + \cot^{-1}x = \frac{\pi}{2}\). Comparing this with the identity for \(y=4\), which is \(\tan^{-1}4 + \cot^{-1}4 = \frac{\pi}{2}\), it is clear by inspection that \(x=4\).

3. If \(|x| \le 1\), then \(2\tan^{-1}x + \sin^{-1}(\frac{2x}{1+x^2})\) is equal to...

For the condition \(|x| \le 1\), the identity \(\sin^{-1}\left(\frac{2x}{1+x^2}\right) = 2\tan^{-1}x\) is valid. Substituting this into the given expression: \[ 2\tan^{-1}x + \sin^{-1}\left(\frac{2x}{1+x^2}\right) = 2\tan^{-1}x + 2\tan^{-1}x = 4\tan^{-1}x \] Answer: (a) \(4\tan^{-1}x\).

Three Marks Questions: LEVEL - 1

1. Prove: \(\tan^{-1}\frac{\cos x}{1-\sin x} - \cot^{-1}\sqrt{\frac{1+\cos x}{1-\cos x}} = \frac{\pi}{4}\)

We simplify the two terms on the LHS. For the first term, we use \(\cos x = \sin(\frac{\pi}{2}-x)\) and \(\sin x = \cos(\frac{\pi}{2}-x)\), followed by half-angle formulas: \[ \frac{\cos x}{1-\sin x} = \frac{\sin(\frac{\pi}{2}-x)}{1-\cos(\frac{\pi}{2}-x)} = \frac{2\sin(\frac{\pi}{4}-\frac{x}{2})\cos(\frac{\pi}{4}-\frac{x}{2})}{2\sin^2(\frac{\pi}{4}-\frac{x}{2})} = \cot\left(\frac{\pi}{4}-\frac{x}{2}\right) \] Using \(\cot\theta = \tan(\frac{\pi}{2}-\theta)\), this is \(\tan(\frac{\pi}{2} - (\frac{\pi}{4}-\frac{x}{2})) = \tan(\frac{\pi}{4}+\frac{x}{2})\). So, \(\tan^{-1}\left(\frac{\cos x}{1-\sin x}\right) = \frac{\pi}{4}+\frac{x}{2}\). For the second term, using half-angle formulas: \[ \cot^{-1}\sqrt{\frac{1+\cos x}{1-\cos x}} = \cot^{-1}\sqrt{\frac{2\cos^2(x/2)}{2\sin^2(x/2)}} = \cot^{-1}(|\cot(x/2)|) \] Assuming \(x/2 \in (0, \pi/2)\), this simplifies to \(\cot^{-1}(\cot(x/2)) = x/2\). Therefore, the LHS becomes: \((\frac{\pi}{4} + \frac{x}{2}) - \frac{x}{2} = \frac{\pi}{4} = \text{RHS}\). Hence proved.

2. Prove that: \(\tan^{-1}\left(\frac{\sqrt{1+x^2}+\sqrt{1-x^2}}{\sqrt{1+x^2}-\sqrt{1-x^2}}\right) = \frac{\pi}{4} + \frac{1}{2}\cos^{-1}x^2\)

Let \(x^2 = \cos(2\theta)\), so \(\theta = \frac{1}{2}\cos^{-1}(x^2)\). Assuming \(x^2 \in [0, 1]\), we can choose \(\theta \in [0, \pi/4]\). The LHS becomes \(\tan^{-1}\left(\frac{\sqrt{1+\cos(2\theta)}+\sqrt{1-\cos(2\theta)}}{\sqrt{1+\cos(2\theta)}-\sqrt{1-\cos(2\theta)}}\right)\). Using \(1+\cos(2\theta) = 2\cos^2\theta\) and \(1-\cos(2\theta) = 2\sin^2\theta\): \[ \text{LHS} = \tan^{-1}\left(\frac{\sqrt{2\cos^2\theta}+\sqrt{2\sin^2\theta}}{\sqrt{2\cos^2\theta}-\sqrt{2\sin^2\theta}}\right) = \tan^{-1}\left(\frac{\cos\theta+\sin\theta}{\cos\theta-\sin\theta}\right) \] Dividing the numerator and denominator by \(\cos\theta\): \[ \text{LHS} = \tan^{-1}\left(\frac{1+\tan\theta}{1-\tan\theta}\right) = \tan^{-1}\left(\tan(\frac{\pi}{4}+\theta)\right) = \frac{\pi}{4}+\theta \] Substituting back \(\theta = \frac{1}{2}\cos^{-1}(x^2)\), we get \(\frac{\pi}{4} + \frac{1}{2}\cos^{-1}x^2 = \text{RHS}\). Hence proved.

3. Prove that: \(\sec^2(\tan^{-1}2) + \csc^2(\cot^{-1}3) = 15\)

We use the Pythagorean identities \(\sec^2\theta = 1+\tan^2\theta\) and \(\csc^2\phi = 1+\cot^2\phi\). Let \(\theta = \tan^{-1}2\) and \(\phi = \cot^{-1}3\). This implies \(\tan\theta = 2\) and \(\cot\phi = 3\). The LHS of the equation is \(\sec^2\theta + \csc^2\phi\). Substituting the identities and values: \[ \text{LHS} = (1+\tan^2\theta) + (1+\cot^2\phi) = (1 + 2^2) + (1 + 3^2) = 5 + 10 = 15 = \text{RHS} \] Hence proved.

4. Find x if \(3\sin^{-1}(\frac{2x}{1+x^2}) - 4\cos^{-1}(\frac{1-x^2}{1+x^2}) + 2\tan^{-1}(\frac{2x}{1-x^2}) = \frac{\pi}{3}\)

We use the identities that relate these expressions to \(2\tan^{-1}x\). Assuming \(0 \le x < 1\), the following hold: \(\sin^{-1}(\frac{2x}{1+x^2}) = 2\tan^{-1}x\), \(\cos^{-1}(\frac{1-x^2}{1+x^2}) = 2\tan^{-1}x\), \(\tan^{-1}(\frac{2x}{1-x^2}) = 2\tan^{-1}x\). Substituting these into the equation: \[ 3(2\tan^{-1}x) - 4(2\tan^{-1}x) + 2(2\tan^{-1}x) = \frac{\pi}{3} \] \[ (6 - 8 + 4)\tan^{-1}x = \frac{\pi}{3} \implies 2\tan^{-1}x = \frac{\pi}{3} \] \[ \tan^{-1}x = \frac{\pi}{6} \implies x = \tan\left(\frac{\pi}{6}\right) = \frac{1}{\sqrt{3}} \] Since \(x = 1/\sqrt{3}\) satisfies the condition \(0 \le x < 1\), this is the valid solution.

LEVEL - 2

1. Prove that: \(\tan(\frac{\pi}{4} + \frac{1}{2}\cos^{-1}\frac{a}{b}) + \tan(\frac{\pi}{4} - \frac{1}{2}\cos^{-1}\frac{a}{b}) = \frac{2b}{a}\)

Let \(\theta = \frac{1}{2}\cos^{-1}\frac{a}{b}\), so \(\cos(2\theta) = \frac{a}{b}\). The LHS becomes \(\tan(\frac{\pi}{4}+\theta) + \tan(\frac{\pi}{4}-\theta)\). Using tangent sum/difference formulas: \[ \text{LHS} = \frac{1+\tan\theta}{1-\tan\theta} + \frac{1-\tan\theta}{1+\tan\theta} = \frac{(1+\tan\theta)^2 + (1-\tan\theta)^2}{(1-\tan\theta)(1+\tan\theta)} \] \[ = \frac{1+2\tan\theta+\tan^2\theta + 1-2\tan\theta+\tan^2\theta}{1-\tan^2\theta} = \frac{2(1+\tan^2\theta)}{1-\tan^2\theta} \] Using the identity \(\cos(2\theta) = \frac{1-\tan^2\theta}{1+\tan^2\theta}\), this is equal to \(\frac{2}{\cos(2\theta)}\). Substituting back \(\cos(2\theta) = a/b\), we get: \[ \text{LHS} = \frac{2}{a/b} = \frac{2b}{a} = \text{RHS} \]

2. Simplify: \(\sin^{-1}[x\sqrt{1-x} - \sqrt{x}\sqrt{1-x^2}]\)

The expression inside the brackets matches the sine subtraction formula, \(\sin(A-B) = \sin A \cos B - \cos A \sin B\). Let \(\sin A = x\) and \(\sin B = \sqrt{x}\). For this to be well-defined, we assume \(x \in [0, 1]\). Then \(A = \sin^{-1}x\) and \(\cos A = \sqrt{1-x^2}\). Also, \(B = \sin^{-1}\sqrt{x}\) and \(\cos B = \sqrt{1-(\sqrt{x})^2} = \sqrt{1-x}\). Substituting these into the original expression: \[ \sin^{-1}[\sin A \cos B - \cos A \sin B] = \sin^{-1}(\sin(A-B)) \] Assuming \(A-B\) lies in the principal value range, this simplifies to \(A-B\). The simplified form is \(\sin^{-1}x - \sin^{-1}\sqrt{x}\).

3. Prove that: \(\cos^{-1}x + \cos^{-1}(\frac{x}{2} + \frac{\sqrt{3-3x^2}}{2}) = \frac{\pi}{3}\)

The argument of the second term is \(\frac{x}{2} + \frac{\sqrt{3}\sqrt{1-x^2}}{2} = x \cdot \frac{1}{2} + \sqrt{1-x^2} \cdot \frac{\sqrt{3}}{2}\). This matches the form of \(\cos(A-B) = \cos A \cos B + \sin A \sin B\). Let \(\cos^{-1}x = A\) and \(\cos^{-1}(1/2) = B\). The expression is \(\cos(A-B)\). The second term is \(\cos^{-1}(\cos(A-B)) = \cos^{-1}(\cos(\cos^{-1}x - \cos^{-1}(1/2)))\). Using the identity \(\cos^{-1}u + \cos^{-1}v = \cos^{-1}(uv - \sqrt{1-u^2}\sqrt{1-v^2})\), a related identity is \(\cos^{-1}u - \cos^{-1}v = \cos^{-1}(uv + \sqrt{1-u^2}\sqrt{1-v^2})\) for \(u \le v\). The problem seems to use a variant. Let's rearrange: \(\cos^{-1}x + \cos^{-1}(\dots) = \pi/3\). The identity holds if the second term is \(\pi/3 - \cos^{-1}x\). This occurs if \(\frac{x}{2} + \frac{\sqrt{3-3x^2}}{2} = \cos(\pi/3 - \cos^{-1}x)\). \(\cos(\pi/3-A) = \cos(\pi/3)\cos A + \sin(\pi/3)\sin A = (1/2)x + (\sqrt{3}/2)\sqrt{1-x^2}\). This is an identity. So the second term is indeed \(\pi/3-\cos^{-1}x\), provided the argument is in the principal range. The LHS becomes \(\cos^{-1}x + (\frac{\pi}{3} - \cos^{-1}x) = \frac{\pi}{3}\). Hence proved.

4. Write the simplest form of \(\cos(2\tan^{-1}\sqrt{\frac{1-x}{1+x}})\)

Let \(x = \cos\theta\), so \(\theta = \cos^{-1}x\). We assume \(x \in (-1, 1]\). The term inside \(\tan^{-1}\) is \(\sqrt{\frac{1-\cos\theta}{1+\cos\theta}} = \sqrt{\frac{2\sin^2(\theta/2)}{2\cos^2(\theta/2)}} = |\tan(\theta/2)|\). For \(x \in (-1, 1]\), \(\theta \in [0, \pi)\), so \(\theta/2 \in [0, \pi/2)\), where \(\tan(\theta/2) \ge 0\). The expression becomes \(\cos(2\tan^{-1}(\tan(\theta/2)))\). Since \(\theta/2\) is in the principal range of \(\tan^{-1}\), this simplifies to \(\cos(2 \cdot \frac{\theta}{2}) = \cos\theta\). As \(x=\cos\theta\), the simplest form is \(x\).

LEVEL - 3

1. Show: \(\cos[\tan^{-1}(\sin(\cot^{-1}x))] = \sqrt{\frac{x^2+1}{x^2+2}}\)

Let \(\cot^{-1}x = \theta\), which implies \(\cot\theta = x\). We can express \(\sin\theta\) in terms of \(\cot\theta\): \[ \sin\theta = \frac{1}{\csc\theta} = \frac{1}{\sqrt{1+\cot^2\theta}} = \frac{1}{\sqrt{1+x^2}} \] The expression now becomes \(\cos\left[\tan^{-1}\left(\frac{1}{\sqrt{x^2+1}}\right)\right]\). Next, let \(\tan^{-1}\left(\frac{1}{\sqrt{x^2+1}}\right) = \phi\), so \(\tan\phi = \frac{1}{\sqrt{x^2+1}}\). We need to find \(\cos\phi\). Using the identity \(\sec^2\phi = 1+\tan^2\phi\): \[ \cos\phi = \frac{1}{\sec\phi} = \frac{1}{\sqrt{1+\tan^2\phi}} = \frac{1}{\sqrt{1 + \left(\frac{1}{\sqrt{x^2+1}}\right)^2}} \] \[ = \frac{1}{\sqrt{1 + \frac{1}{x^2+1}}} = \frac{1}{\sqrt{\frac{x^2+1+1}{x^2+1}}} = \sqrt{\frac{x^2+1}{x^2+2}} \] Thus, the original expression simplifies to \(\sqrt{\frac{x^2+1}{x^2+2}}\). Hence proved.

2. If \(\cos^{-1}\frac{x}{a} + \cos^{-1}\frac{y}{b} = \alpha\), then prove that \(\frac{x^2}{a^2} - \frac{2xy}{ab}\cos\alpha + \frac{y^2}{b^2} = \sin^2\alpha\).

We are given \(\cos^{-1}\frac{x}{a} + \cos^{-1}\frac{y}{b} = \alpha\). Applying the identity \(\cos^{-1}A + \cos^{-1}B = \cos^{-1}(AB - \sqrt{1-A^2}\sqrt{1-B^2})\), we get: \[ \cos^{-1}\left(\frac{xy}{ab} - \sqrt{1-\frac{x^2}{a^2}}\sqrt{1-\frac{y^2}{b^2}}\right) = \alpha \] Taking the cosine of both sides yields: \[ \frac{xy}{ab} - \sqrt{\left(1-\frac{x^2}{a^2}\right)\left(1-\frac{y^2}{b^2}\right)} = \cos\alpha \] Rearranging the terms to isolate the radical and then squaring both sides: \[ \left(\frac{xy}{ab} - \cos\alpha\right)^2 = \left(1-\frac{x^2}{a^2}\right)\left(1-\frac{y^2}{b^2}\right) \] \[ \frac{x^2y^2}{a^2b^2} - \frac{2xy}{ab}\cos\alpha + \cos^2\alpha = 1 - \frac{x^2}{a^2} - \frac{y^2}{b^2} + \frac{x^2y^2}{a^2b^2} \] Canceling the \(\frac{x^2y^2}{a^2b^2}\) term and rearranging, we have: \[ \frac{x^2}{a^2} + \frac{y^2}{b^2} - \frac{2xy}{ab}\cos\alpha = 1 - \cos^2\alpha \] Using the identity \(\sin^2\alpha = 1 - \cos^2\alpha\), we arrive at the desired result: \[ \frac{x^2}{a^2} - \frac{2xy}{ab}\cos\alpha + \frac{y^2}{b^2} = \sin^2\alpha \] Hence proved.

3. If \(\cos^{-1}p + \cos^{-1}q + \cos^{-1}r = \pi\), then show that \(p^2+q^2+r^2+2pqr = 1\).

Given \(\cos^{-1}p + \cos^{-1}q + \cos^{-1}r = \pi\), we can write: \[ \cos^{-1}p + \cos^{-1}q = \pi - \cos^{-1}r \] Applying the identity for the sum of inverse cosines to the left side: \[ \cos^{-1}(pq - \sqrt{1-p^2}\sqrt{1-q^2}) = \pi - \cos^{-1}r \] Taking the cosine of both sides and using the fact that \(\cos(\pi - \theta) = -\cos\theta\): \[ pq - \sqrt{(1-p^2)(1-q^2)} = \cos(\pi - \cos^{-1}r) = -\cos(\cos^{-1}r) = -r \] Isolating the radical term and squaring both sides: \[ (pq + r)^2 = (\sqrt{(1-p^2)(1-q^2)})^2 \] \[ p^2q^2 + 2pqr + r^2 = (1-p^2)(1-q^2) \] \[ p^2q^2 + 2pqr + r^2 = 1 - p^2 - q^2 + p^2q^2 \] Canceling the \(p^2q^2\) term and rearranging gives the required result: \[ p^2 + q^2 + r^2 + 2pqr = 1 \] Hence proved.

ASSERTION-REASON

1. Assertion (A): The range of \(f(x)=2\sin^{-1}x + \frac{3\pi}{2}\) is \([\frac{\pi}{2}, \frac{5\pi}{2}]\). Reason (R): The range of \(\sin^{-1}(x)\) is \([0, \pi]\).

The principal value range of \(\sin^{-1}x\) is \([-\frac{\pi}{2}, \frac{\pi}{2}]\). Therefore, we have the inequality: \(-\frac{\pi}{2} \le \sin^{-1}x \le \frac{\pi}{2}\). Multiplying by 2 gives: \(-\pi \le 2\sin^{-1}x \le \pi\). Adding \(\frac{3\pi}{2}\) throughout: \(-\pi + \frac{3\pi}{2} \le 2\sin^{-1}x + \frac{3\pi}{2} \le \pi + \frac{3\pi}{2}\).
