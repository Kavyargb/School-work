1. Variables & Data Types

Topic: Introduction to variables for storing data and the fundamental data types in Python.
Introduction to Python

Python is a high-level, interpreted, general-purpose programming language known for its simple syntax and readability. It allows you to write programs with fewer lines of code compared to other languages like C++ or Java. This makes it an excellent choice for beginners and for rapid application development.

Variables

A variable is a named location in memory used to store data. You can think of it as a labeled box where you can put information. In Python, you create a variable by assigning a value to it using the equals sign (`=`).

# Variable assignment
course_name = "Python Basics"
chapter = 1
is_completed = False

Naming Rules: Variable names must start with a letter or an underscore (`_`), can contain letters, numbers, and underscores, and are case-sensitive (`age`, `Age`, and `AGE` are three different variables).

Core Data Types

Python has several built-in data types. The most fundamental are:

  • Integer (`int`): Whole numbers, both positive and negative, without decimals. Ex: `10`, `-3`, `0`.
  • Float (`float`): Numbers with a decimal point. Ex: `9.81`, `-0.5`, `3.14`.
  • String (`str`): A sequence of characters, enclosed in single (`'`) or double (`"`) quotes. Ex: `"Hello, World!"`, `'Python'`.
  • Boolean (`bool`): Represents one of two values: `True` or `False`. Used for logical operations.

You can check the type of a variable using the built-in `type()` function.

number = 42
print(type(number))  # Output: 

pi = 3.14
print(type(pi))      # Output:
Type Casting

You can convert data from one type to another using casting functions like `int()`, `float()`, `str()`, and `bool()`.

age_str = "25"
age_int = int(age_str) # Converts string to integer
print(age_int + 5)    # Output: 30

2. Operators & Expressions

Topic: Using operators to perform calculations and comparisons, forming expressions.

Operators are special symbols that perform operations on values and variables (operands). An expression is a combination of values, variables, and operators that evaluates to a single value.

Arithmetic Operators

Used for mathematical calculations.

  • `+` (Addition), `-` (Subtraction), `*` (Multiplication)
  • `/` (Division): Always returns a float. `10 / 3` is `3.333...`
  • `//` (Floor Division): Divides and rounds down to the nearest whole number. `10 // 3` is `3`.
  • `%` (Modulus): Returns the remainder of a division. `10 % 3` is `1`.
  • `**` (Exponentiation): Raises to the power of. `2 ** 3` is `8`.
Comparison Operators

Used to compare two values. They return a Boolean (`True` or `False`).

  • `==` (Equal to)
  • `!=` (Not equal to)
  • `>` (Greater than), `<` (Less than)
  • `>=` (Greater than or equal to), `<=` (Less than or equal to)
x = 10
y = 5
print(x > y)      # Output: True
print(x == y * 2) # Output: True
Logical Operators

Used to combine conditional statements.

  • `and`: Returns `True` if both operands are true.
  • `or`: Returns `True` if at least one operand is true.
  • `not`: Inverts the boolean value.
has_key = True
knows_password = False
print(has_key and knows_password) # Output: False
print(has_key or knows_password)  # Output: True
print(not has_key)                  # Output: False
Assignment Operators

Used to assign values to variables. The standard is `=`, but compound operators provide a shorthand.

`count += 1` is a shorthand for `count = count + 1`. This works for other operators as well (`-=`, `*=`, `/=`, etc.).

3. Control Structures

Topic: Controlling the flow of program execution with conditional statements and loops.

Control structures allow you to execute certain blocks of code conditionally or repeatedly.

Conditional Statements: `if`, `elif`, `else`

These structures allow your program to make decisions. The code inside an `if` block runs only if its condition is `True`. `elif` (else if) checks another condition if the previous `if`/`elif` was `False`. `else` runs if all preceding conditions were `False`.

score = 85
if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B") # This block will run
else:
    print("Grade: C or below")
Loops

Loops are used to execute a block of code multiple times.

`for` Loop

The `for` loop iterates over a sequence (like a list, tuple, string, or a range of numbers).

# Looping through a range of numbers
for i in range(5):  # range(5) generates numbers 0, 1, 2, 3, 4
    print(i)

# Looping through a string
for char in "Python":
    print(char, end="-") # Output: P-y-t-h-o-n-
`while` Loop

The `while` loop continues to execute as long as its condition remains `True`.

count = 0
while count < 3:
    print("Looping...")
    count += 1
Loop Control: `break` and `continue`
  • `break`: Immediately exits the current loop.
  • `continue`: Skips the rest of the current iteration and proceeds to the next one.

4. Strings

Topic: Manipulating text data using Python's powerful string methods.

A string is an immutable sequence of characters. Immutability means that once a string is created, it cannot be changed. Methods that appear to modify a string actually return a new string with the modification.

Indexing and Slicing

You can access individual characters using an index (starting from 0) or a range of characters using slicing `[start:stop:step]`.

s = "Hello, World!"
print(s[0])          # Output: H
print(s[-1])         # Output: ! (negative indexing from the end)
print(s[7:12])      # Output: World (slicing)
Common String Methods
  • len(s): Returns the length of the string.
  • s.upper() / s.lower(): Returns a new string in all uppercase or all lowercase.
  • s.strip(): Returns a new string with leading/trailing whitespace removed.
  • s.startswith(prefix) / s.endswith(suffix): Returns `True` or `False`.
  • s.replace(old, new): Returns a new string where all occurrences of `old` are replaced by `new`.
  • s.split(delimiter): Splits the string at the delimiter and returns a list of substrings.
  • delimiter.join(list_of_strings): Joins elements of a list into a single string, with the delimiter between each element.
  • s.find(substring): Returns the index of the first occurrence of the substring, or -1 if not found.
message = "  Welcome to Python!  "
print(message.strip())         # Output: "Welcome to Python!"
print(message.lower().startswith('welcome')) # Output: True (method chaining)
words = message.strip().split(' ')
print(words)                # Output: ['Welcome', 'to', 'Python!']
Formatted Strings (f-strings)

F-strings provide a concise and convenient way to embed expressions inside string literals.

name = "Alice"
age = 30
greeting = f"Hello, my name is {name} and I am {age} years old."
print(greeting)

5. Lists

Topic: Understanding and using lists, Python's versatile, mutable, ordered sequence type.

A list is an ordered and mutable collection of items. Mutable means you can change its content—add, remove, or modify elements—after it has been created. Lists are defined by enclosing comma-separated items in square brackets `[]`.

Figure: Lists are modified in-place, while operations on tuples create new objects.

Common List Methods
  • len(my_list): Returns the number of items in the list.
  • my_list.append(item): Adds an item to the end of the list.
  • my_list.extend(another_list): Appends all items from another list.
  • my_list.insert(index, item): Inserts an item at a specific index.
  • my_list.remove(item): Removes the first occurrence of an item. Raises an error if the item is not found.
  • my_list.pop(index): Removes and returns the item at the given index. If no index is specified, it removes and returns the last item.
  • my_list.sort(): Sorts the list in place.
  • my_list.reverse(): Reverses the elements of the list in place.
  • my_list.index(item): Returns the index of the first occurrence of an item.
  • my_list.count(item): Returns the number of times an item appears in the list.
numbers = [1, 5, 3, 4, 2]
numbers.sort()     # numbers is now [1, 2, 3, 4, 5]
numbers.append(6)  # numbers is now [1, 2, 3, 4, 5, 6]
popped_item = numbers.pop(2) # popped_item is 3, numbers is [1, 2, 4, 5, 6]

6. Tuples

Topic: Introducing tuples, the immutable counterpart to lists, and their use cases.

A tuple is an ordered and immutable collection of items. Once a tuple is created, you cannot change its contents. Tuples are defined by enclosing comma-separated items in parentheses `()`.

Their immutability makes them faster than lists and suitable for protecting data from accidental changes. They can also be used as keys in dictionaries, whereas lists cannot.

point = (10, 20, 30)
print(point[0]) # Output: 10

# The following line would cause an error:
# point[0] = 15  # TypeError: 'tuple' object does not support item assignment
Tuple Methods

Tuples have very few methods due to their immutability.

  • my_tuple.count(item): Returns the number of times an item appears.
  • my_tuple.index(item): Returns the index of the first occurrence of an item.
Tuple Packing and Unpacking

You can create a tuple by "packing" values together. The reverse, "unpacking," allows you to assign the items of a tuple to multiple variables at once.

coordinates = (45.6, 90.2) # Packing
lat, lon = coordinates    # Unpacking
print(f"Latitude: {lat}, Longitude: {lon}")

This is very useful for swapping variables: `a, b = b, a`.

7. Dictionaries

Topic: Working with dictionaries, which store data in key-value pairs for efficient lookup.

A dictionary is a mutable, unordered (in Python < 3.7) or insertion-ordered (Python 3.7+) collection of key-value pairs. Each key must be unique and immutable (e.g., a string, number, or tuple). Dictionaries are optimized for retrieving values when the key is known. They are defined with curly braces `{}`.

student = {
    "name": "Alice",
    "age": 21,
    "courses": ["Math", "Physics"]
}
Accessing, Modifying, and Adding Items
  • Accessing: Use square brackets with the key: `student['name']`. This raises a `KeyError` if the key doesn't exist.
  • Modifying: Assign a new value to an existing key: `student['age'] = 22`.
  • Adding: Assign a value to a new key: `student['major'] = "Computer Science"`.
Common Dictionary Methods
  • d.get(key, default): Safely gets a value. Returns `None` (or the specified `default` value) if the key is not found, instead of raising an error.
  • d.keys(): Returns a view object displaying a list of all keys.
  • d.values(): Returns a view object displaying a list of all values.
  • d.items(): Returns a view object displaying a list of key-value tuple pairs. This is useful for looping.
  • d.pop(key): Removes the specified key and returns the corresponding value.
  • d.update(other_dict): Updates the dictionary with key-value pairs from another dictionary.
# Safely getting a value
major = student.get('major', 'Undeclared')
print(major) # Output: Undeclared

# Looping through items
for key, value in student.items():
    print(f"{key}: {value}")

8. Sets

Topic: Using sets for storing unique elements and performing mathematical set operations.

A set is a mutable, unordered collection of unique elements. They are useful for membership testing and eliminating duplicate entries. Sets are defined with curly braces `{}` or the `set()` function. An empty set must be created with `set()`, as `{}` creates an empty dictionary.

numbers = {1, 2, 3, 2, 1}
print(numbers) # Output: {1, 2, 3} (duplicates are automatically removed)
Adding and Removing Elements
  • my_set.add(item): Adds a single element to the set.
  • my_set.remove(item): Removes an element. Raises a `KeyError` if the element is not in the set.
  • my_set.discard(item): Removes an element if it is present, but does nothing otherwise (no error).
  • my_set.pop(): Removes and returns an arbitrary element from the set.
  • my_set.clear(): Removes all elements from the set.
Set Operations

Sets excel at mathematical operations.

  • Union: `set1 | set2` or `set1.union(set2)`. Returns a new set with all elements from both sets.
  • Intersection: `set1 & set2` or `set1.intersection(set2)`. Returns a new set with only the elements common to both sets.
  • Difference: `set1 - set2` or `set1.difference(set2)`. Returns a new set with elements in `set1` but not in `set2`.
  • Symmetric Difference: `set1 ^ set2` or `set1.symmetric_difference(set2)`. Returns a new set with elements in either set, but not both.
a = {'apple', 'banana', 'cherry'}
b = {'banana', 'orange', 'mango'}

# Intersection
print(a & b) # Output: {'banana'}
# Union
print(a | b) # Output: {'apple', 'cherry', 'mango', 'orange', 'banana'}

Solution

Float (`float`). The standard division operator `/` in Python always returns a float, even if the result is a whole number (e.g., `5.0`).

Integer (`int`). The floor division operator `//` performs division and rounds the result down to the nearest whole number, returning an integer.

Q49. Fix the error in this code:

print("Next year, you will be " + (age + 1))
Solution

The input() function returns a string. You cannot add a string and an integer. You must first cast the input to an integer and then cast the result back to a string for concatenation.

age_str = input("Enter your age: ")
age_int = int(age_str)
print("Next year, you will be " + str(age_int + 1))
Q50. What is the final value of count? 
count = 0
for i in range(10):
    if i % 2 == 0:
        continue
    count += 1
Solution

The final value of count is 5. The loop iterates for `i` from 0 to 9. The `continue` statement skips the `count += 1` line for all even numbers (0, 2, 4, 6, 8). The count is only incremented for the odd numbers (1, 3, 5, 7, 9), which are 5 numbers in total.
